# awesome-cli
A curated list of modern unix cli tools for daily life

## Editors

 - [Neovim](https://neovim.io): a modern version of vim with better configuration options
 - [helix](https://helix-editor.com/): a modern modal editor (vim-like) with multiple cursors in mind
 - [Lunar VIM](https://helix-editor.com/): a neovim distribution with easier installation

## CLI Convenience
 - [bottom](https://github.com/ClementTsang/bottom): a terminal process and system monitor
 - [mcfly](https://github.com/cantino/mcfly): a fast history browser with fuzzy search
 - [fzf](https://github.com/junegunn/fzf): fast fuzzy searcher cli
 - [prmpt](): a faster prompt. allows quicker prompt loading than many shells
